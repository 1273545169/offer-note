### 题目描述

输入两个单调递增的链表，输出两个链表合成后的链表，当然我们需要合成后的链表满足单调不减规则。

### 思路解析

非递归法：
```python
class Solution:
    # 返回合并后列表
    def Merge(self, pHead1, pHead2):
        if not pHead1:
            return pHead2

        if not pHead2:
            return pHead1

        node = root = ListNode(0)
        while pHead1 and pHead2:
            if pHead1.val < pHead2.val:
                node.next = ListNode(pHead1.val)
                pHead1 = pHead1.next
            else:
                node.next = ListNode(pHead2.val)
                pHead2 = pHead2.next
            node = node.next

        if pHead1:
            node.next = pHead1
        if pHead2:
            node.next = pHead2

        return root.next

```

递归法：


```python
class Solution:
    # 返回合并后列表
    def Merge(self, pHead1, pHead2):
        if not pHead1:
            return pHead2

        if not pHead2:
            return pHead1

        if pHead1.val < pHead2.val:
            pHead1.next = self.Merge(pHead1.next, pHead2)
            return pHead1
        else:
            pHead2.next = self.Merge(pHead1, pHead2.next)
            return pHead2
```
