### 题目描述

我们可以用2*1的小矩形横着或者竖着去覆盖更大的矩形。请问用n个2*1的小矩形无重叠地覆盖一个2*n的大矩形，总共有多少种方法？

### 思路解析

f(n)=f(n-1)+f(n-2)

```python

class Solution:
    def rectCover(self, number):
        if number == 0:
            return 0
        count, a, b = 0, 0, 1
        while count < number:
            a, b = b, a + b
            count += 1
            if count == number:
                return b

```
